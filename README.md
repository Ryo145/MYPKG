# mypkg
このリポジトリは千葉工業大学先進工学部未来ロボティクス学科４sのロボットシステム学の講義で使用したものである。  

# インストール方法
ROS2を使用できる環境でこのリポジトリをクローンして使用する。
以下でクローン出来る。

```
$ git clone https://github.com/Ryo145/mypkg.git
```


# 必要なソフトウェア
* ROS2  
* Python
 
# テスト環境
* Ubuntu 22.04.2 LTS
